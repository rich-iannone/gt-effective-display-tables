# Creating Effective Display Tables with the gt Package

[![Render and Publish](https://github.com/riannone/gt-effective-display-tables/actions/workflows/quarto-publish.yml/badge.svg)](https://github.com/riannone/gt-effective-display-tables/actions/workflows/quarto-publish.yml)

This is the repository for the book "Creating Effective Display Tables with the gt Package", a comprehensive guide to building professional, publication-ready tables in R using the **gt** package.

## About this book

The **gt** package provides a powerful and flexible system for creating display tables in R. This book teaches you how to use **gt** effectively, from basic table construction to advanced customization techniques.

**Read the book online:** https://riannone.github.io/gt-effective-display-tables/

## What you'll learn

- understanding table structure and **gt**'s component-based approach
- transforming raw values into reader-friendly formats
- applying colors, fonts, borders, and spacing for aesthetic tables
- adding footnotes, source notes, and conditional formatting
- incorporating sparklines, small multiples, and summary statistics
- tailoring tables for different output formats (HTML, PDF, Word, PowerPoint)
- learning best practices for effective table design
